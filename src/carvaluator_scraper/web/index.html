Ce inseamna rezultatele?

Explicatii scurte pentru termenii folositi in evaluare si in comparatia modelelor.

01

Estimarea finala ponderata

Este pretul principal afisat de CarValuator. Aplicatia combina estimarile mai multor modele si acorda mai multa importanta celor care au avut erori mai mici la testare.

02

MAE

Eroarea medie absoluta. Daca MAE este 2.500 EUR, modelul greseste in medie cu aproximativ 2.500 EUR, indiferent daca estimeaza prea mult sau prea putin. Mai mic este mai bine.

03

R2

Arata cat de bine explica modelul diferentele dintre preturi. O valoare mai apropiata de 1 este mai buna. Nu inseamna procent de precizie si nu garanteaza o estimare perfecta pentru fiecare masina.

04

RMSE

O alta masura a erorii, care penalizeaza mai puternic greselile foarte mari. Este utila pentru a observa modelele care pot rata serios pretul unor masini neobisnuite.

05

MAE + acord intre modele

Porneste de la MAE, dar reduce influenta unui model daca estimarea lui este foarte departe de restul. Este varianta mai prudenta atunci cand un singur algoritm produce o valoare extrema.

06

KNN si masini similare

KNN cauta exemple apropiate ca marca, model si specificatii. Lista de masini similare ajuta la verificarea rezultatului, dar preturile lor nu sunt adaugate din nou in estimarea finala.

07

Toleranta verdictului

Este marja acceptata in jurul pretului estimat. La o toleranta de 10%, un anunt aflat cu cel mult 10% peste sau sub estimare este considerat la pret corect.

08

Verdictul

Pret corect inseamna ca anuntul este in marja aleasa. Pret prea mare este peste limita superioara, iar pret suspect de mic este sub limita inferioara si merita verificari suplimentare.
